Bienvenue sur Ecrion Project

Bienvenue sur ExilBatch

Un projet conçu pour offrir une portabilité presque totale, tout en laissant place à vos idées !

Installation rapide ou portable

Exécutez l’installateur, cliquez sur Next à chaque étape, puis choisissez la langue. Et voilà, le tour est joué ! Pour trouver ExilBatch‑Écrion, tapez Ecrion dans la barre de recherche.

Besoin d'aide ?

Si vous voulez activer ou désactiver des fonctionnalités, allez dans le menu ExilBatch-Écrion et cliquez sur le bouton (Panel de …). Vous aurez alors des options que vous pourrez modifier. Note : nous avons ajouté un deuxième IDE, car le premier ne charge pas les fichiers assez vite.

A propos !

ExilBatch a été créé pour être un environnement de développement avec une syntaxe personnalisée. le but est de rendre le developpement plus simple ! le projet comprend un IDE, un compiler, un editeur d'himage, un interpreteur, un terminal... (Le batch n'ai pas limité c'est ce que l'on en fait qui le limit !)
